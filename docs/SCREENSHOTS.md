## Interactive Examples

### Example 1: Enterprise Network
```python
# Generate 100-device enterprise network
topology = generator.generate_topology(
    network_type="enterprise",
    num_routers=5,
    num_switches=15,
    num_hosts=80,
    security_level="high"
)
```

**Result:** 100 devices, 250+ links, 15 network segments

### Example 2: Data Center
```python
# Create spine-leaf data center
topology = generator.generate_topology(
    network_type="datacenter",
    num_routers=4,  # Spine switches
    num_switches=8,  # Leaf switches
    num_hosts=100,  # Servers
    redundancy=True
)
```

**Result:** High-availability data center with full redundancy

### Example 3: Hybrid Cloud
```python
# AWS hybrid cloud integration
hybrid = cloud_builder.create_hybrid_topology(
    on_premise=topology,
    cloud_provider="aws",
    integration_type="site-to-site vpn",
    bandwidth=10000  # 10 Gbps
)
```

**Result:** Secure hybrid cloud with VPN connectivity
